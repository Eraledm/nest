## Descripción

Nest es un framework para crear aplicaciones <a href="http://nodejs.org" target="_blank">Node.js</a> eficientes y escalables del lado del servidor. Hace uso de JavaScript moderno, está construido con <a href="http://www.typescriptlang.org" target="_blank">TypeScript</a> (mantiene compatibilidad con JavaScript puro) y combina elementos OOP (Programación Orientada a Objetos), FP (Programación Funcional), y FRP (Programación Funcional Reactiva).

<p>En el fondo, Nest hace uso de <a href="https://expressjs.com/" target="_blank">Express</a>, pero también ofrece compatibilidad con un gran rango de otras librerias, por ejemplo <a href="https://github.com/fastify/fastify" target="_blank">Fastify</a>, permitiendo un uso fácil de innumerables plugins de terceros que estén disponibles.</p>

## Filosofía

<p>En los últimos años, gracias a Node.js, JavaScript se ha convertido en el “lingua franca” de la web para aplicaciones tanto de frontend como backend, dando lugar a proyectos increíbles como <a href="https://angular.io/" target="_blank">Angular</a>, <a href="https://github.com/facebook/react" target="_blank">React</a> y <a href="https://github.com/vuejs/vue" target="_blank">Vue</a> los cuales mejoran la productividad del desarrollador y permiten la construcción de aplicaciones frontend rápidos, testeables y extensibles. Sin embargo, mientras que del lado del servidor hay muchas librerías, helpers y herramientas magníficas para Node, ninguno de ellos ofrece una solución efectiva al problema principal - la arquitectura.</p>
<p>Nest tiene como objetivo proporcionar una arquitectura de aplicaciones lista para usar que permita la creación, sin esfuerzo, de aplicaciones altamente testeables, escalables, poco acopladas y fáciles de mantener. La arquitectura está fuertemente inspirada en Angular.</p>
